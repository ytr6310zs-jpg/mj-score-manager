# スクリーンショット・画像ルール
- 画像フォルダ: `docs/images/` に格納
- 命名規則: `step-01-login.png`, `step-02-dashboard.png` のように手順番号先頭で統一
- 形式: PNG 推奨（可読性優先）。Retina対策で原寸の2x画像を用意しておくと印刷時に鮮明
- キャプションと alt: すべての画像に短いキャプション（タイトル）と alt テキストを必ず付与
  - 例: `![ログイン画面 — ユーザー名入力欄](docs/images/step-01-login.png "ログイン画面")`
- 注釈: 矢印・囲み・番号を使って注釈。色だけで情報を示さない（色盲対応）
