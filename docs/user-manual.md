# 概要
本マニュアルは、本アプリを日常的に利用するエンドユーザー（大会運営者、スコア入力者、閲覧者）向けの操作手順をまとめています。

# 前提条件
- 対応ブラウザ: Chrome / Firefox / Edge（いずれも最新の安定版を推奨）
- インターネット接続があること
- 利用に必要なアカウント（メールアドレス／パスワード）と適切な権限

# ログイン
- 目的: アプリにログインしてダッシュボードへ入る
- 前提: 有効なアカウント
- 手順:
  1. 管理者から通知されたアプリURLにアクセスします
  2. 画面の「ログイン」ボタンをクリックします
  3. メールアドレスとパスワードを入力して「ログイン」をクリックします
  4. MFA（有効な場合）は表示に従い認証コードを入力します
- 期待結果: ダッシュボード（対局一覧等）が表示されます
- 失敗時の対処: パスワードリセットを実行し、それでも不可なら管理者へ連絡してください

# ダッシュボードの見方
- 対局一覧: 最近の対局が一覧表示されます。各行から「編集」「削除」が可能です（権限依存）。
- フィルタ: 期間・プレイヤーなどで表示を絞れます。

# プレイヤー管理
- 目的: 新しいプレイヤーの登録・編集・削除
- 前提: 管理者権限が必要な場合があります（画面に表示される権限情報を確認）
- 手順（新規登録）:
  1. ナビゲーションで「Players」を開きます
  2. 「Add player」ボタンをクリックします
  3. 名前、ニックネーム、必要項目を入力して「保存」をクリックします
- 期待結果: プレイヤー一覧に登録済みの行が追加されます

# 対局（スコア）操作
- 目的: 対局を新規登録し、スコアを保存する
- 前提: 登録するプレイヤーが事前に存在していること
- 手順（新規対局）:
  1. 「Matches」を開き「New match」をクリックします
  2. 日時、参加プレイヤーを選択します
  3. 各プレイヤーのスコア（点数または順位）を入力します
  4. 必要に応じて Yakuman や備考を追加します
  5. 「保存」をクリックします
- 編集/削除: 対局の「編集」ボタンで修正、または「削除」→確認ダイアログで実行します
- 注意: 入力形式や必須項目は画面のバリデーションメッセージに従ってください

# 統計／ランキングの参照
- 目的: 任意期間の集計結果やランキングを見る
- 手順:
  1. 「Stats」を開きます
  2. 期間フィルタやプレイヤーフィルタを設定します
  3. 表示されたテーブルやグラフを確認します
- 期待結果: フィルタに応じた集計結果が表示されます

# CSV エクスポート
- 目的: 対局データをCSVで保存する
- 手順:
  1. 対局一覧またはエクスポート画面で期間を設定します
  2. 「CSVエクスポート」ボタンをクリックします
  3. ダウンロードされたファイルを保存します
- 出力される主な項目（例）: 対局日時, プレイヤー名, スコア, 備考

# 管理者向け機能（概要）
- Yakuman 管理、バルクインポート、バックアップ/復元などの操作は管理者用画面に記載されています。詳細は管理者にお問い合わせください。

# トラブルシューティング（代表例）
- ログインできない
  - 対処: パスワードリセット → 別ブラウザ／プライベートモードで試行 → 管理者へ連絡
- 対局が保存できない
  - 対処: 必須項目の入力確認 → ページ再読み込み → エラーメッセージのスクリーンショットを添えて管理者へ連絡

# サポートに報告する際の必須情報
- 発生日時、実行した操作、スクリーンショット（可能ならコンソールログ）、影響を受けるアカウント
