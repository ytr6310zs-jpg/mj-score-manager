# 概要
本マニュアルは、本アプリを日常的に利用するエンドユーザー（大会運営者、スコア入力者、閲覧者）向けの操作手順をまとめています。

# 前提条件
- 対応ブラウザ: Chrome / Firefox / Edge（いずれも最新の安定版を推奨）
- インターネット接続があること
- 利用に必要なアカウント（メールアドレス／パスワード）と適切な権限

# MFA設定
- 目的: セキュリティ強化
- 前提: アクセス情報フォルダを閲覧できること
- 手順: 
  - GoogleAuthenticatorアプリをインストール
  - Googleドライブの「アクセス情報」を開く
  - セットアップキーを登録する場合
    - ワンタイムパスワード/セットアップキー.txt の内容をコピー
    - GoogleAuthenticatorを起動し、右下の「+」ボタン → セットアップキーを入力
      - コード名：任意（麻雀成績入力）
      - 鍵：さきほどのセットアップキー
      - 鍵の種類：時間ベース
    - 追加をタップ
    - コード名に指定した情報が追加される
  - QRコードを登録する場合（別端末に登録する場合）
    - アクセス情報フォルダの ワンタイムパスワード/totp_prd_QRコード.png を開く
    - GoogleAuthenticatorを起動し、右下の「+」ボタン → QRコードをスキャン
    - 「MahjonScoreManager:MahjongAdmin」が追加される

# ログイン
- 目的: アプリにログインしてダッシュボードへ入る
- 前提: アクセス情報フォルダを閲覧できること
- 手順:
  1. 以下のリンク or アクセス情報フォルダを開きアプリURLからアクセスします
     - [MahjonScoreManager](https://mj-score-manager-tfvp.vercel.app/login)  
  ![ログイン画面](images/step-01-login.png)
  1. 共通パスワード、GoogleAuthenticatorのワンタイムパスワードを入力
  2. 画面の「ログイン」ボタンをクリックします

# スコア入力操作
- 目的: 対局を新規登録し、スコアを保存する
- 前提: 特になし
- 手順（新規対局）:
  1. 「スコア入力」を開く  
  ![score-input](./images/step-02-score-input.png)
  2. 日時、参加プレイヤーを選択します
     - プレーヤーが未登録の場合、プレーヤー選択欄に名前を入力し新規追加が可能  
     ![new-player](images/step-03-new-player.png)
  3. 各プレイヤーのスコア、飛び/飛ばし、焼き鳥を入力します
  4. 役満の登録
     1. プレーヤー、役満種別を選択し登録をタップ  
     ![yakuman-registration](images/step-04-yakuman-registration.png)
     2. 続けて役満を登録するには、同様にプレーヤー、役満種別を選択し登録をタップ
     - 発生した役満種別が存在しない場合、管理 > 役満種別から登録できます
  5. 「保存」をクリックします
- 注意: 入力形式や必須項目は画面のバリデーションメッセージに従ってください

# 対局履歴一覧
- 目的: 登録済みの対局を確認する
- 前提: 対局が登録済み
- 手順: 
  1. 対局履歴を開く  
  ![match-list](images/step-05-match-list.png)
  1. 対局を検索する  
  ![search-match](images/step-06-search-match.png)
     - 今年 を選択し絞込みするとその年の対局が表示されます
     - 当日 を選択し絞込みすると当日の対局が表示されます
     - 任意 を選択すると指定した期間で絞込んだ対局が表示されます  
     ![search-match-range](images/step-07-search-match-range.png)
- 表示している対局情報を画面下部のCSV出力からダウンロードできます  
![export-matches](images/step-09-export-matches.png)

# 対局編集
- 目的: 登録済みの対局を編集する
- 前提: 対局が登録済み
- 手順: 
  1. 対局履歴一覧と同様に対局を検索
  2. 編集する対局の編集ボタンをタップ
  3. 情報を修正したのち、対局を編集をタップ  
  ![edit-match](images/step-08-edit-match.png)

# 対局削除
- 目的: 登録済みの対局を削除する
- 前提: 対局が登録済み
- 手順:
  1. 対局履歴一覧と同様に対局を検索
  2. 対局の削除リンクをタップ
- 注意: 削除した対局は復元できないため注意してください

# 成績集計
- 目的: 指定した期間のランキングを表示
- 前提: 対局が登録済み
- 手順:
  1. 成績集計を開く  
  ![stats](images/step-10-stats.png)
  1. 集計範囲を指定する
    - 今年 を選択し絞込みするとその年の対局が表示されます
    - 当日 を選択し絞込みすると当日の対局が表示されます
    - 任意 を選択すると指定した期間で絞込んだ対局が表示されます  
- PCなど横幅の広い端末で表示すると成績を確認しやすいです  
![stats-landscape](images/step-12-stats-landscape.png)
- 表示している集計を下部のCSV出力からダウンロードできます  
![export-stats](images/step-11-export-stats.png)

# 管理メニュー
- 目的: システムに必要な情報の管理を行う
- 前提: データ修正が必要な状況
- 手順:
  1. 右上の「≡」をタップ  
  ![admin-link](images/step-13-admin-link.png)
  1. 管理メニューが表示される  
  ![admin-menu](images/step-14-admin-menu.png)
- 注意: システムの動作にかかわるため、必要な場合のみアクセスしてください

# 管理）プレイヤー管理
- 目的: 新しいプレイヤーの登録・編集・削除
- 前提: プレーヤーのデータ修正が必要な状況
- 手順: 
  1. 管理メニューを開き、プレイヤー管理をタップ  
  ![players](images/step-15-players.png)
  1. プレーヤーの追加、編集（名前の修正）、削除が行えます
- 注意: 削除すると復元できませんので注意して操作してください

# 管理）役満種別管理
- 目的: 新しい役満種別の登録・編集・削除
- 前提: 新しい役満種別の登録・修正が必要な状況
- 手順: 
  1. 管理メニューを開き、役満種別管理をタップ  
  ![yakuman-type](images/step-16-yakuman-type.png)
  1. 役満種別の追加、編集、削除が行えます
     - コードは重複しないものを登録してください
     - 順序は役満登録時の並び順を表します
     - 状態を無効にすると役満登録時に表示されません 
- 注意: 削除すると復元できませんので注意して操作してください

# 管理者向け機能（概要）
- Yakuman 管理、バルクインポート、バックアップ/復元などの操作は管理者が実施します。

# トラブルシューティング（代表例）
- ログインできない
  - 対処: パスワードリセット → 別ブラウザ／プライベートモードで試行 → 管理者へ連絡
- 対局が保存できない
  - 対処: 必須項目の入力確認 → ページ再読み込み → エラーメッセージのスクリーンショットを添えて管理者へ連絡

# サポートに報告する際の必須情報
- 発生日時、実行した操作、スクリーンショット（可能ならコンソールログ）、影響を受けるアカウント
