# images フォルダ

- 保存場所: `docs/images/`
- 命名規則: `step-01-短い説明.png` のように先頭に手順番号を付ける
- 形式: PNG 推奨（可読性優先）。必要に応じて高解像度版（@2x）を併存させる
- キャプション: Markdown 内で必ずキャプション（title）と alt テキストを付与する
- 例:
  - `docs/images/step-01-login.png`
  - `docs/images/step-01-login@2x.png`（Retina用）

注: 画像はコミット時に最適化して容量を抑えること。大きな画像は `assets/images/` 等で管理する運用も検討する。
