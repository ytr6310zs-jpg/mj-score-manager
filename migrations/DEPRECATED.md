# `migrations/` — DEPRECATED

このディレクトリはレガシー経路です。リポジトリのマイグレーション適用の正本は現在 `supabase/migrations/` です。

目的
- 過去に `node-pg-migrate` 等で使っていたマイグレーションがここに残っています。

現状の扱い
- 今後の運用では `migrations/` を使用しません。
- 当面はアーカイブ（read-only）扱いとし、内容を削除せず保管します。

移行手順（推奨）
1. `migrations/` にあるスキーマ変更を `supabase/migrations/` にタイムスタンプ付き SQL ファイルとして移行する。
2. 各移行ファイルについて PR を作成し、CI の検証（バックアップ → supabase db push → 検証）を通す。
3. 移行完了後、`migrations/` を削除またはリポジトリ外へアーカイブする。削除は明確な PR で行うこと。

問い合わせ
- 変更の流れに不明点がある場合は PR コメントで担当者をアサインしてください。
