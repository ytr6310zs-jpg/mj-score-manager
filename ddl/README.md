# ddl/ ディレクトリについて

この `ddl/` ディレクトリは参照用のスキーマ定義スナップショットを格納します。日常のスキーマ変更や CI での適用ソースとしては扱わず、read-only として運用してください。

ガイドライン
- 新しいスキーマ変更は `supabase/migrations/` にタイムスタンプ付き SQL ファイルを追加して適用します。`ddl/` へ直接追加・編集しないでください。
- 既存 `ddl/` の内容は参照可能ですが、実運用の正本は `supabase/migrations/` です。

例
- 役満テーブルの追加が必要な場合: `supabase/migrations/20260402000000_create_yakuman_occurrences.sql` を作成し PR を作成してください。
